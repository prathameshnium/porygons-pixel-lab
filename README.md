ğŸš€ How to Use

Clone the repository:

git clone [https://github.com/YourUsername/porygons-pixel-lab.git](https://github.com/YourUsername/porygons-pixel-lab.git)
cd porygons-pixel-lab


Install dependencies:
It's recommended to use a virtual environment.

python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate
pip install -r requirements.txt


(Note: You will need to create a requirements.txt file listing packages like manim, matplotlib, etc.)

Run a script:
Navigate to a project directory and execute the script.

For Manim:

cd manim/token_embedding
manim -pqm token_embedding.py TokenEmbeddingScene


For Matplotlib:

cd matplotlib/simple_line_plot
python line_plot.py
