This lab primarily runs on Python and a few key libraries:

Manim: For creating complex mathematical animations and concept explainers.

Matplotlib: For robust, publication-quality 2D and 3D data plots.

FreeCAD: For parametric 3D modeling, with Python scripting capabilities.

Seaborn: For high-level statistical data visualization.

Custom Python Scripts: Various self-made tools and helper functions to automate and streamline figure creation.

ğŸš€ How to Use

Clone the repository:

```bash
git clone https://github.com/YourUsername/porygons-pixel-lab.git
cd porygons-pixel-lab
```

Install dependencies:
It's recommended to use a virtual environment.

```bash
python -m venv venv
source venv/bin/activate  # On Windows, use: .\venv\Scripts\activate
pip install -r requirements.txt
```
(Note: A `requirements.txt` file should be created to list all necessary packages like `manim`, `matplotlib`, etc.)

Run a script:
Navigate to a project directory and execute the script.

For Manim:
```bash
cd manim/token_embedding/
manim -pql token_embedding.py TokenEmbeddingScene
```

For Matplotlib:
```bash
cd matplotlib/simple_line_plot/
python line_plot.py
```
